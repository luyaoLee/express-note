# express-note
[线上预览](http://47.91.156.35:9981)

## 使用方法
```
# 克隆仓库
git clone git@github.com:luyaoLee/express-note.git
# cd 到仓库目录后安装依赖
npm i
# 启动本地服务器，打开 http://localhost:3000
npm start
# 当然你也可以在./bin/www文件里修改端口
```
